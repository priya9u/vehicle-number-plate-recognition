# vehicle-number-plate-recognition# Vehicle Number Plate Recognition

This project detects and recognizes vehicle number plates using image processing techniques.

## 🚀 Features
- Detects number plates from images
- Extracts text using recognition techniques
- Can be used for traffic monitoring and security

## 🛠 Tech Used
- Python
- OpenCV
- OCR techniques
